# GameBot
discord bot that provides information about games

## Commands

### Default prefix is `~` for commands

  * `~help` - provides a list of commands and provides detailed information about any command

  * `~reload` - reloads a command

### Dead by Daylight Commands

  * `~dbddlc` - lists Dead by Daylight DLC and allows users to add DLC to their collection

  * `~perkroulette` - randomizes four perks

  * `~shrineofsecrets` - gets the latest shrine of secrets

## Installation

### Prerequisites

- `npm`

### Local Setup

- `git clone https://github.com/allentruong72/GameBot.git`

- `npm install`

- `create a config.json`
```javascript
{
    "prefix": "~",
    "token": "inserttokenhere"
}
```

- `npm run start`
